USAGE:
------

In your Python code:

from prediction_api import predict_single_row

# Your data
row = {
    'tcp': 0.0,
    'udp': 0.0,
    # ... all 49 features
}

# Get prediction
result = predict_single_row(row, model_name='ensemble')
print(result['interpretation'])  # 'Normal' or 'Attack'
print(result['confidence'])      # 0.0 to 1.0
